Ubuntu 16.04.6 LTS (Xenial Xerus)

1) Download opencv: ./download_opencv.sh
2) Build opencv: ./build_opencv.sh
3) Build: ./build.sh 
4) Run: Go to "build_debug" or "build_release" directory and launch ./image_viewer d=</your_directory_path/>. For example, ./image_viewer -d=/home/sergey/image_viewer/test_images/.
